# moonkit

Claude Code skills for work automation.

## Skills

### work-log

GitHub, Slack, Jira, Google Calendar, Notion에서 활동을 수집해서 업무일지/성과보고서/이력서 소재를 만들어주는 스킬.

- **journal** — 주간 업무일지
- **impact** — 역량 기반 성과 보고서
- **organize** — 사내 보고용 or 이력서용 성과 정리

자세한 내용은 [skills/work-log/](skills/work-log/) 참고.
